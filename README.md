# Izhan Akhter - Professional Video Editor Portfolio

A modern, responsive portfolio website for a professional video editor with a cinematic dark theme.

## Features

- **Fully Responsive Design**: Optimized for mobile, tablet, and desktop
- **Modern UI/UX**: Clean typography with smooth animations and transitions
- **Portfolio Showcase**: Grid layout for video projects with filtering capabilities
- **Skills Section**: Visual progress bars for editing tools proficiency
- **Contact Form**: Working contact form with Vercel serverless function integration
- **SEO Optimized**: Proper meta tags and semantic HTML structure
- **Performance Focused**: Optimized images and efficient code

## Tech Stack

- HTML5
- CSS3 (Flexbox/Grid)
- Vanilla JavaScript
- Font Awesome Icons
- Google Fonts
- Vercel for deployment

## Deployment on Vercel

1. Push the code to a GitHub repository
2. Sign in to [Vercel](https://vercel.com)
3. Click "New Project" and import your GitHub repository
4. Vercel will automatically detect the configuration and deploy
5. The contact form will work out of the box with the provided serverless function
